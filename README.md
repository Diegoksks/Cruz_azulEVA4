# Cruz Azul ERP - EVA4

Proyecto de arquitectura AWS para Farmacias Cruz Azul.

## Servicios considerados

- EC2 Web ERP
- Node.js + Express
- PostgreSQL dockerizado
- Application Load Balancer
- Auto Scaling Group
- CloudWatch
- S3 para respaldos y objetos
- Grupos de seguridad
- VPC con subredes públicas y privadas

## Ruta de trabajo

/srv/cruz_azul-erp/

## Objetivo

Implementar una maqueta funcional segura, escalable y monitoreada en AWS para el aplicativo Web ERP.
